The problem

Manual QA tests one path at a time.

  • Real users diverge — they fuzz inputs, double-click, paste payloads.
  • Each tester explores one branch; the rest of the tree never gets walked.
  • Bugs that need concurrent or adversarial behavior almost always slip past.

What slips through

races, XSS, validation bypass, broken UI state, server crashes, auth holes — all need someone to actively try to break it.

The idea

Fork the browser. Run every “what if” in parallel.

One snapshot of the page. The model proposes 2–5 adversarial intents based on what's actually rendered. Each intent gets its own headless Chromium context, driven by a vision agent. They all run at the same time.

How it works

Snapshot · Fork · Diff

1. Snapshot

Open the target URL once. Capture screenshot + DOM + storage state. This is the shared starting point.

2. Fork

GPT-4o-mini reads the page and writes 2–5 adversarial intents. Each spawns its own headless browser context, resumed from the snapshot.

3. Diff

The control fork must succeed. Any fork that triggers a dialog, a 5xx, duplicates, or the agent's own bug verdict gets flagged.

Inside one fork

Screenshot → LLM → action → repeat.

  • Take a JPEG of the page + a DOM snippet.
  • Send to GPT-4o-mini with the intent: “break the inputs”, “inject XSS”, “cause a race”.
  • Model returns one of: click · fill · press · eval · spawn · done.
  • Playwright executes. Loop, up to 5 steps.
  • spawn recursively forks again — up to 3 levels deep, 22 forks per run.

Verdict signals

XSS — a JS dialog fired
5xx — server returned 500+
RACE — N items created when 1 was expected
PASS — control fork completed cleanly

What the operator sees

A live tree of every fork, streaming.

  • Every fork is a node — screenshots stream in via Chrome DevTools Protocol, ~5 fps.
  • Bug forks turn red. Hover for the evidence: dialog count, 5xx count, duplicate rows.
  • Click into a bug node → expanded view with the agent's step-by-step thoughts and replay scrubber.
  • One-click “Fix with Claude” generates a prompt with the failing intent + bug evidence.
